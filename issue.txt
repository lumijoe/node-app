## リロード作業の効率化
nodemonの導入
1：package.json追加
2：npm install -g nodemon
3：npm install --save-dev nodemon
4：package.jsonのscriptsでnodemonで起動できるようにテキストを修正
    "scripts": {
        "dev": "nodemon app.js"
    },
5：npm run devで起動実行が可能になる（nodemon導入後）
